```sh
cd minimam-ts-starter
npm i
```

## ファイルの操作

- ファイルの実行方法

```sh
# main.tsを実行する場合
npm run main

# 指定のパスにあるファイルを実行する場合
npm run tsx <指定のパス>
```

- テストの実行方法

```sh
npm run test
```
